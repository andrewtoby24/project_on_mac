Xiaomi 14 Ultra

Description

The Xiaomi 14 Ultra comes with 6.73-inch AMOLED display with 120Hz refresh rate and Qualcomm Snapdragon 8 Gen 3 processor. Specs also include 5000mAh battery with 90W charging speed and Quad camera setup on the back.

PROS

  • High refresh rate (120Hz)
  • 5G Ready
  • High screen-to-body ratio (90.18 %)
  • Supports wireless charging
  • High pixel density screen (521 PPI)
